Engine Integration

Exporting to your engine

Unity (PNG 16-bit)

Use Export, choose PNG 16-bit. Note: Unity's native Import Raw doesn't work with PNG — instead, use a script or the Terrain Tools package's Import Heightmap feature.

Unreal Engine (EXR or PNG)

Export as EXR 32-bit for maximum precision. In Unreal: Landscape Editor → Import → select the EXR file. Make sure Height Type is set to Float32.

Godot (PNG 16-bit)

Export PNG 16-bit. In Godot: create a HeightMapShape3D and use the map_data property with a script to load the PNG via Image.load().
